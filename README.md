# EAPES

Connect EAPES to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/eapes) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-eapes@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-eapes --url https://cortex.joai.ai/mcp/apps/eapes
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/eapes.git ~/.cursor/plugins/local/joai-eapes
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-eapes&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktZWFwZXMiOnsidXJsIjoiaHR0cHM6Ly9jb3J0ZXguam9haS5haS9tY3AvYXBwcy9lYXBlcyJ9fX0=) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/eapes
```
